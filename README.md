# 🧑‍💻 Developer Portfolio Builder

A React application that generates a live developer portfolio as you fill out a form. Built as a beginner-to-intermediate React project covering all core React concepts.

## ✨ Features

- 📝 **Live Preview** — portfolio updates instantly as you type
- 🎨 **Skills Tags** — add and remove skills with one click
- 🎓 **Education Section** — add multiple education entries
- 💼 **Projects Section** — showcase your projects with GitHub links
- 🔗 **Social Links** — GitHub, LinkedIn, Portfolio website
- 🌙 **Dark / Light Mode** — toggle between themes
- ⬇️ **PDF Download** — export your portfolio as a PDF instantly
- 💾 **localStorage** — data is saved even after page refresh
- 🗑️ **Clear Data** — reset everything with one click

---

## 🧠 React Concepts Covered

| Concept | Where Used |
|---|---|
| `useState` | Every form input field |
| `useEffect` | Saving data to localStorage |
| Props | Passing data from App → Form → Portfolio |
| Conditional Rendering | Sections appear only when filled |
| Lists & Keys | Skills, education, projects with `.map()` |
| Controlled Inputs | Every input tied to state |
| Component Reusability | Form and Portfolio components |
